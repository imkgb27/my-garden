#### 프론트엔드 vs. 백엔드
- Front-end
	- "사용자 화면"
	- 사용자가 웹 브라우저에서 보는 모든 시각적 요소
	- UI (User-Interface), UX (User-eXperience)
	- GUI (Graphic User Interface), CLI (Command Line Interface)
- Back-end
	- "서버와 데이터"
	- 사용자에게 보이지 않는 뒤편에서 데이터를 저장, 가공, 전달하는 시스템을 관리
- Full-stack 개발자: 둘 다 다룰 수 있는 개발자

#### API (Application Programming Interface)
- 서로 다른 소프트웨어 애플리케이션이 상호 통신하고 데이터를 교환할 수 있도록 하는 정의와 규칙의 집합.
- 주로 *데이터 요청*과 *응답* 방식으로 작동
- 프론트엔드와 백엔드 간 통신 접점 역할
